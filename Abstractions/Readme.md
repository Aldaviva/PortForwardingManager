This library contains a shared interface to let you write a plugin for [PortForwardingService](https://github.com/Aldaviva/PortForwardingManager).

Your plugin project should depend on this package.

- [Documentation with examples](https://github.com/Aldaviva/PortForwardingManager/wiki/Plugin-Authoring)
- [Sample plugin source code](https://github.com/Aldaviva/PortForwardingManager/tree/master/SamplePlugin)
